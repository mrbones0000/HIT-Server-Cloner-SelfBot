**Discord Server Cloner (Selfbot)**
Aviso Importante

Este proyecto es solo con fines educativos y de aprendizaje.
El uso de selfbots va en contra de los Términos de Servicio de Discord.

El autor no se hace responsable del uso indebido de este software ni de posibles sanciones como baneos o suspensiones de cuenta.

Usa este proyecto únicamente en entornos privados o para fines de estudio.

Descripción

Discord Server Cloner es una herramienta desarrollada en JavaScript (Node.js) que permite clonar la estructura completa de un servidor de Discord utilizando una cuenta de usuario (selfbot).

El sistema replica automáticamente la estructura del servidor original, manteniendo el orden y la configuración de los elementos.

Características

Clonado completo de categorías

Clonado de canales de texto

Clonado de canales de voz

Copia de roles y permisos

Conserva el orden original del servidor

Automatización completa del proceso


Requisitos


Una cuenta de Discord

Token de usuario (selfbot)
